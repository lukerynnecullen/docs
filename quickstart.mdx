## Get Started in 3 Steps

<Steps>
  <Step title="Generate an API Key">
    Log in to [precital.com](https://precital.com) and navigate to your account settings. Generate a new API key from the API Keys section.
  </Step>
  <Step title="Check Your Credits">
    Verify you have credits available for API operations.
  </Step>
  <Step title="Make Your First Request">
    Use the example below to analyze an audio file.
  </Step>
</Steps>

## Example: Analyze an Audio File

Here's a simple example to get you started:

<CodeGroup>
```bash cURL
curl -X POST https://api.precital.com/v1/audio-analyses \
  -H "X-API-Key: your-api-key-here" \
  -H "Content-Type: application/json" \
  -d '{
    "audio_url": "https://example.com/track.mp3",
    "track_title": "My Amazing Song",
    "artist_name": "Artist Name"
  }'
```

```javascript JavaScript
const response = await fetch('https://api.precital.com/v1/audio-analyses', {
  method: 'POST',
  headers: {
    'X-API-Key': 'your-api-key-here',
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    audio_url: 'https://example.com/track.mp3',
    track_title: 'My Amazing Song',
    artist_name: 'Artist Name'
  })
});

const data = await response.json();
console.log(data);
```

```python Python
import requests

response = requests.post(
    'https://api.precital.com/v1/audio-analyses',
    headers={
        'X-API-Key': 'your-api-key-here',
        'Content-Type': 'application/json'
    },
    json={
        'audio_url': 'https://example.com/track.mp3',
        'track_title': 'My Amazing Song',
        'artist_name': 'Artist Name'
    }
)

print(response.json())
```
</CodeGroup>

## Check Your Credits

Before making requests, check your credit balance:

<CodeGroup>
```bash cURL
curl -X GET https://api.precital.com/v1/credits \
  -H "X-API-Key: your-api-key-here"
```

```javascript JavaScript
const response = await fetch('https://api.precital.com/v1/credits', {
  headers: {
    'X-API-Key': 'your-api-key-here'
  }
});

const data = await response.json();
console.log(data);
```
</CodeGroup>
